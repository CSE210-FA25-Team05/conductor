# Meeting Notes

This directory contains all meeting notes for the project.

## Structure

```
meeting_notes/
├── all_hands_notes/      # All-hands team meetings
└── ta_meeting_notes/      # TA meetings
```

## Format Guidelines

### File Naming

- Use format: `meeting_type_MMDDYY.md` or `meeting_type_MMMdd.md`
- Examples:
  - `ta_meeting_notes_102025.md`
  - `ta_meeting_notes_oct27.md`
  - `all_hands_01_102025.md`

### Document Structure

```markdown
# Meeting Type - Meeting Notes

## Meeting # (Date)

### Topic 1

- Point 1
- Point 2

### Topic 2

- Discussion point
- Action item

### Action Items

- [ ] Task 1
- [ ] Task 2
```

### Required Information

- Date of meeting
- Attendees (optional but recommended)
- Main topics discussed
- Decisions made
- Action items
- Questions raised and answers

## Guidelines

1. **Document every meeting** - All team, TA, and sub-team meetings must be documented
2. **Store in GitHub** - All meeting notes should be committed to this repository
3. **Be clear and concise** - Focus on key decisions and action items
4. **Include context** - Note who said what when relevant
